First Step: One time setup
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

Second Step:Start a Repo
git init                  
git clone <url>     

Third Step:Working with Files
git status              
git add <file>           
git add .                
git commit -m "message"  

Fourth Step: Branching
git branch              
git branch <name>        
git checkout <name> 
git checkout -b <name>  
git merge <branch>


Fifth Step: Connect to Remote
git remote add origin<url>
git remote -v
